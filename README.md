# Dog Fact Saver

**Submitted by:** Dwarakesh Baraneetharan (dbaranee)

**Group Members:**
*   Dwarakesh Baraneetharan (dbaranee)

**App Description:**
This application allows users to enter their name to fetch a random dog fact from an external API and save it to a MongoDB database. Users can view all saved facts on the homepage.
